# service-discovery
Service Discovery library that extends Apache Curator framework with etcd support

## Maven

```xml
<dependency>
    <groupId>com.daedafusion</groupId>
    <artifactId>service-discovery</artifactId>
    <version>1.0</version>
</dependency>
```
